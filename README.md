# 老罗的安驾课程

这是一个可直接发布到 GitHub Pages 的静态网页，内容分成两部分：

- Part 1：核心原则与细节点
- Part 2：逐节追加的课程档案

当前只记录了 Lesson 1，后续上到 Lesson 2、Lesson 3 后再继续追加，不预填未上课程。

## 本地查看

直接打开 `index.html` 即可查看。

## 发布到 GitHub Pages

把本目录里的文件放到 GitHub 仓库根目录，进入仓库的 `Settings -> Pages`，选择从默认分支发布即可。

如果你的仓库已经有其他内容，也可以把这些文件放进 `docs/` 目录，然后在 GitHub Pages 设置里选择 `docs/` 作为发布目录。

## 移动端适配

页面已经做成响应式布局。同一个网址在电脑、平板、手机上会自动根据屏幕宽度调整展示，不需要单独做一个手机版。

## 后续追加课程

新增课程时，继续把新材料整理成 Lesson 2、Lesson 3 即可，不需要预先填空白课程。

建议每节课材料按下面格式继续记录：

```text
lesson2
练习内容1 ...
练习内容2 ...
老师强调 ...
自己容易犯错 ...
下次复盘提醒 ...
```
