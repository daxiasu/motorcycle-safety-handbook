# 老罗 RoadCraft：路上功夫

`RoadCraft` 可以理解为“道路技艺”“路上功夫”，指向安全驾驶里的路线判断、风险感知和控车手艺。放在 `老罗 RoadCraft：路上功夫` 里，`Road` 也和“罗”形成一点自然的声音勾连：不是硬造梗，而是品牌语义和中文名字刚好咬住。

## 品牌宣言

**老罗 RoadCraft：路上功夫 = 克制的机械美学 + 真实训练的粗粝感 + 安全驾驶的秩序感。**

我们的母题是 **Controlled Force / 被控制的力量**。

速度存在，但不失控。力量存在，但被训练、被纪律、被判断驯服。

## 设计语言

- Road Line：用弯道线、刹车线、视线方向线作为视觉基因。
- Control Points：把头盔、视线、手、核心、腿、刹车做成人车关系的控制节点。
- Black / Gold / Asphalt：黑来自柏油、皮革和机件阴影；金来自仪表、卡钳和金属铭牌。
- Engineer’s Calm：少解释，少堆砌，用秩序、比例和留白建立高级感。
- Training Is The Luxury：真正贵的不是炫技，而是被训练过的判断力。

## 页面结构

这是一个可直接发布到 GitHub Pages 的静态网页，当前一级菜单包括：

- 骑行法则：从每次训练里持续提炼出来的安全驾驶与控车原则
- 训练轨迹：逐节追加的课程记录，公开版只保留训练主题、动作要点和复盘方向
- 教官老罗：老罗老师介绍与宣传素材

当前已记录 Lesson 1、0630 复习课、Lesson 2、Lesson 3（含 Part 2）和 Lesson 4，后续上到 Lesson 5 后再继续追加，不预填未上课程。

## 本地查看

直接打开 `index.html` 即可查看。

## 发布到 GitHub Pages

把本目录里的文件放到 GitHub 仓库根目录，进入仓库的 `Settings -> Pages`，选择从默认分支发布即可。

如果你的仓库已经有其他内容，也可以把这些文件放进 `docs/` 目录，然后在 GitHub Pages 设置里选择 `docs/` 作为发布目录。

## 移动端适配

页面已经做成响应式布局。同一个网址在电脑、平板、手机上会自动根据屏幕宽度调整展示，不需要单独做一个手机版。

## 后续追加课程

新增课程时，继续把新材料整理成 Lesson 5、Lesson 6 即可，不需要预先填空白课程。

如果某个要点被老师反复强调，或者跨多个练习都适用，就从对应 Lesson 里抽出来，补到骑行法则里。

## 训练素材与隐私

本仓库是 public 项目，不提交可识别真人照片、原始训练视频、车牌、位置信息或其他个人隐私素材。

如需在页面里展示训练画面，优先使用以下方式：

- 使用脱敏后的局部图、背影图、低识别度素材。
- 使用朋友绘制的品牌视觉、示意图或非真人素材。
- 个人复盘照片和视频保存在本地私密目录，不进入 Git 仓库。

`.gitignore` 已忽略 `assets/training/`，避免误提交个人训练素材。
